IMP
-----------------------------------------------------------
[ 機能 ]
論理含意を返します。

[ 書式 ]
<数式1> IMP <数式2>

[ 文例 ]
PRINT -15 IMP 7
 [表示] 15

[ 解説 ]
<数式1> <数式2> を 2 進数にした際のビット単位で処理されます。
符号ビットも含まれます。

    <数式1> <数式2>  結果
       0       0       1
       0       1       1
       1       0       0
       1       1       1
