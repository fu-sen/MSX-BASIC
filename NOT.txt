NOT
-----------------------------------------------------------
[ 機能 ]
ビット反転した値を返します。

[ 書式 ]
NOT <数式>

[ 文例 ]
PRINT 5,-5
 [表示] -6      4

[ 解説 ]
<数式> を 2 進数にした際のビット単位で反転した値を返します。
符号ビットを含めて処理されるため、
結果として符号を反転し -1 した値となります。

    <数式>  結果
       0      1
       1      0

NOT 0 は -1、NOT -1 は 0 となり、この反転動作で使用します。
