￥・₩・＼
-----------------------------------------------------------
[ 機能 ]
除算の商 (整数) を返します。

[ 書式 ]
<割る数> ￥ <割られる数> (日本)
<割る数> ₩ <割られる数> (韓国)
<割る数> ＼ <割られる数> (日本・韓国以外)

[ 文例 ]
PRINT -5 ￥ 3 (日本)
PRINT -5 ₩ 3 (韓国)
PRINT -5 ＼ 3 (日本・韓国以外)
 [表示] -1

[ 解説 ]
CHR$(92) のキャラクターは日本の MSX は ￥、韓国の MSX は ₩、他の MSX は ＼ です。
FIX(<割る数>/<割られる数>) と同じ値になります。
※ INT はマイナスを含む場合に異なる結果を得ます。
